Nextall - React Multivendor Ecommerce Script

Introduction

Welcome to the documentation for Nextall, a React Multivendor Ecommerce Script built with Next.js and MongoDB. This script is designed to provide a robust platform for building a multivendor ecommerce website with a modern frontend and a powerful backend.

Product Details

Nextall is divided into two main applications:

  • Frontend: Developed with Next.js 14 using the App Router and MUI v5.
  • Backend: Developed with Node.js, Express, MongoDB, and Mongoose.

Frontend Configuration

The frontend application uses two environment files for configuration: .env.development and .env.production. 

BASE_URL=
STRIPE_PUBLIC_KEY=
STRIPE_SECRET_KEY=
PAYPAL_CLIENT_ID=
CLOUDINARY_CLOUD_NAME=
BASE_CURRENCY=
SHIPPING_FEE=
JWT_SECRET=

Backend Configuration

The backend application is configured using an .env file. 

MONGODB_URI=
PORT=
RECEIVING_EMAIL=
EMAIL_PASSWORD=
SHIPPING_FEE=
BASE_CURRENCY=
STRIPE_SECRET_KEY=
CLOUDINARY_PUBLISHABLE_KEY=z
CLOUDINARY_SECRET_KEY=
CLOUDINARY_CLOUD_NAME=
CURRENCY=
# COMMISSION in percent
COMMISSION=
JWT_SECRET=

Setup and Installation

Frontend Setup

  1. Open the frontend in the terminal
  2. Install dependencies: npm install
  3. Create and configure the .env.development and .env.production files as shown above.
  4. Start the development server: npm run dev

Backend Setup

  1. Open the backend in the terminal
  2. Install dependencies: npm install
  3. Create and configure the .env file as shown above.
  4. Start the server: npm start

Usage

After setting up both the frontend and backend applications, you can access the ecommerce platform by navigating to the BASE_URL specified in your frontend .env file. From here, you can register as a vendor, add products, and manage orders.
